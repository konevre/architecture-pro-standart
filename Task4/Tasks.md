System Tasks Backlog (High-Level)
Подготовка систем к MVP депозитов, кредитов и подключению кол-центров к ставкам
## 1. Сервис ставок (Rates Service)
Эпики

Создание сервиса ставок как единого источника истины.

Обеспечение доступа к ставкам для всех каналов и кол-центров.

Экспорт ставок для партнёрского кол-центра.

Крупные задачи

 Проектирование доменной модели ставок (депозиты/кредиты, диапазоны сумм, сроки, спецставки).

 Реализация RatesAPI (REST):

GET /rates (общие ставки)

GET /rates/personalized

Admin CRUD (бэк-офис депозитов)

 Веб-UI для бэк-офиса депозитов для редактирования и утверждения ставок.

 История изменений ставок (аудит).

 RatesExportJob: формирование JSON-файла по контракту.

 Защищённая передача JSON партнёрскому КЦ (почта/B2B-портал, защищённые вложения).

 Ведение версий JSON-контракта.

 Мониторинг, алерты, логирование ошибок выгрузки.

## 2. Система кол-центра банка
Эпики

Интеграция со ставками.

Обновление сценариев консультаций.

Крупные задачи

 Модуль RatesClient для вызова RatesAPI.

 Настройка авторизации/токенов для API.

 Локальный кэш ставок + TTL (уменьшение нагрузки).

 Встроить ставки в сценарии CallScriptsEngine (подстановки в подсказки).

 Изменить UI оператора — блок со ставками в карточке клиента.

 Нагрузочное тестирование (пиковые звонки + запросы к API).

 Обучение операторов и методические материалы.
